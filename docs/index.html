クイックスタート（最短）

  1. Discord Developer PortalでBotを作成し、Tokenを取得する
  2. MESSAGE CONTENT INTENT をONにする
  3. ConsoNanceをダウンロードして起動し、Tokenと音声デバイスを選ぶ
  4. Discordのテキストチャンネルで @Bot join #ボイスチャンネル名

※ 詳細は「新規導入ガイド」を参照してください。

注意と協力のお願い

  • OSについて: ConsoNanceはOSを問わず利用できることを目指していますが、現在このドキュメントはWindows版の説明が中心です。
